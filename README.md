# From Scratch

In this repo I tried to implement machine learning algorithms from scratch using only numpy, the main purpose is to show the underluying math rather than memorizing library code.

Understanding the math is essential for real world problems and it adds more flexebility to your skills .

The code was **fully vectorized**, i wanted to use numpy allover the notebooks and also to show that it's handy easy to implement and actually more efficient.

---

## What each notebook contains

- **Concept explanation** — what the algorithm is, when to use it, and the intuition behind it
- **Math** — the key equations written in LaTeX, derived before any code is written
- **Clean implementation** — class-based, readable, consistent structure across all notebooks
- **Visualization** — decision boundaries, cost curves, cluster plots, feature importances
